ISO image builds:

We certify our images using SHA256 and take them from GitHub, even on the official website.
We've fully opened up the ISO image build process with all the necessary configurations on GitHub, and we've also included instructions on the GitHub Wiki.

Repository:

The system uses official Debian repositories with additional settings for availability, freshness, and compatibility. For similar purposes, we also have our own open, moderated repository, which prioritizes maintainers from official software developers or those close to them.
